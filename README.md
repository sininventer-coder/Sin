# StudyHub React Native App

## Project Overview
StudyHub is a mobile application designed to facilitate collaborative learning for students. It provides features such as study groups, resource sharing, and interactive learning materials.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/sininventer-coder/Sin.git
   cd Sin
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the app**:
   - For iOS:
     ```bash
     npx react-native run-ios
     ```
   - For Android:
     ```bash
     npx react-native run-android
     ```

## Features
- **Study Groups**: Create or join study groups to collaborate with peers.
- **Resource Sharing**: Share notes, documents, and other learning materials.
- **Interactive Learning**: Engage with quizzes and interactive content to enhance knowledge retention.
- **Notification System**: Receive important updates and reminders from your study group.
